# EmotionPredictor
EmotionSense AI is an NLP-based machine learning web application that predicts emotions from user text inputs. Built using Python, Streamlit, and Scikit-learn, the project uses text preprocessing, vectorization, and a trained Logistic Regression model to classify emotions like joy, sadness, anger, fear, love, and surprise.
# EmotionSense AI

EmotionSense AI is a machine learning and Natural Language Processing (NLP) project that predicts emotions from text input. The application analyzes user-provided sentences and classifies them into emotions such as Joy, Sadness, Anger, Fear, Love, and Surprise.

The project was built to understand the complete NLP workflow, including text preprocessing, feature extraction, model training, and deployment using Streamlit.

---

## Features

* Emotion prediction from text
* NLP preprocessing pipeline
* Real-time predictions
* Machine learning-based classification
* Streamlit integration

---

## Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* Joblib
* NLP

---

## Machine Learning Workflow

The project includes:

* Text cleaning and preprocessing
* Stopword removal
* Text vectorization using Bag of Words
* Model training using Logistic Regression
* Saving trained models using Joblib
* Real-time prediction through a Streamlit application

---

## Supported Emotions

* Joy
* Sadness
* Anger
* Fear
* Love
* Surprise
